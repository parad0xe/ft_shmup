ft_shmup is a terminal-based shooter game built with ncurses.  
Move your ship, dodge enemies, and shoot to survive as long as possible!

---

## INSTALLATION

Make sure you have the required dependencies:

```bash
sudo apt install make gcc libncurses-dev
```

Clone the project:

```bash
git clone https://github.com/parad0xe/ft_shmup.git
cd ft_shmup
```

Compile the project:

```bash
make
```

---

## USAGE

Run the game with:

```bash
./ft_shmup
```

---

## CONTROLS

* **W** : Move up  
* **A** : Move left  
* **S** : Move down  
* **D** : Move right  
* **Space** : Shoot  
* **P** : Pause / Resume  
* **R** : Restart after Game Over  
* **Q** : Quit the game

__Tip:__ Movement uses classic "WASD" controls for smooth navigation.
